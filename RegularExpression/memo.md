# 雑記

## ○○を含む、AND（かつ）、△△を含まない

```regexp
^(?=.*○○)(?!.*△△).*$
```

## ○○かつ△△

```regexp
^(?=.*○○)(?=.*△△).*$
```

## ○○または△△

```regexp
○○|△△
```

## 改行コードだけ

```regexp
^([*\n])
```

## CR・LF・CR+LFをすべて検索する場合

```regexp
\r\n|\n|\r
```
