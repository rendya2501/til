### 2問目

①売掛金期末残高について差額補充法により、3%の貸倒引当金を設定する。  
→  
2000の3%は60。既に貸倒引当金が40あって、最終的に60にするので貸倒引当金を20足す。  
貸倒引当金繰入 20 / 貸倒引当金 20  

②期末商品有高は500である。なお「仕入」勘定で売上原価を算定する。  
→  
仕入400 / 繰越商品400  
繰越商品500 / 仕入500  

③建物について耐用年数30年、残存価格300の定額法で減価償却している。  
→  
3000/30 = 100  
減価償却費100 / 減価償却累計額100  

④借入金は当期の2月1日に仮り入れたものであり、借り入れから1年後に元本と利息を一括返済、年利4%で借り入れたものである。  
→  
300 * 0.04 = 12  
1か月1,2月と3月で2ヵ月なので2。  
×未払利息2 / 支払利息2  
→  
支払利息2 / 未払利息2  
逆だった・・・。  
**支払利息**という**費用**の発生と**未払利息**という**負債**の発生  
| 借方 | 貸方        |
| :--- | :---------- |
| 資産 | 負債 純資産 |
| 費用 | 収益        |

### 貸借対照表と損益計算書を作る

これがまた面倒臭いんだ。
①三分法の場合、「繰越商品」は「商品」とする。  
②貸倒引当金はマイナスの資産なので、資産側に書いてマイナス表記にする。  
　また、このマイナスな売掛金に対するマイナスなので売掛金の下に書く。  
　試験だと▲表示になっているかも。これはマイナス表示を意味する。  
③「建物減価償却累計額」も対象となる資産からマイナスするので「建物」の下に書く。  
④「未払利息」を忘れずに。  
⑤「当期純利益」も「繰越利益剰余金」とみなせるので足してよい。  

⑥「売上」は「売上高」という表示にする。  
　三分法では「売上」が該当するので。  
⑦売上から仕入を引くので、「仕入」は「売上原価」とする。
