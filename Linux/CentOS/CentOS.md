# CentOSに関する色々

※CentOSはRedHat系  

- OS確認コマンド  
  `$ cat /etc/redhat-release`

## Web開発環境について

`CentOS 7`の場合  

- chrome update コマンド  
`$ sudo yum install https://dl.google.com/linux/direct/google-chrome-stable_current_x86_64.rpm`  

- chrome version 確認コマンド  
`$ google-chrome --version`
