# カレントブランチをpushするコマンド

```git
git fetch
git checkout <ブランチ名>
```

fetchはリモートリポジトリの情報をローカルに持ってくるだけで、実際にワークツリーには反映されていません。  
そこからcheckoutすることで、ローカルにブランチを生成する。  
つまり、リモートのブランチをローカルに持ってくるというわけ。
