# アルゴリズム

プログラミングの中ではなく、アルゴリズムフォルダ単体で作っても十分なレベルだが、
もっと充実するまではここで細々とやっていく。

## 最近見つかったシンプルなソート法

``` C#
var array = new int[2,4,5,6,4,6,8,2,8,9,4];

for(var i = 0 ; array.length; i++){
    for(var j = 0 ; array.length; j++){
        if (array[j] < array[i]){
            swap(array[i],array[j]);
        }
    }
}

// O(n^2)
```
