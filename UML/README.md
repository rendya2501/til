## is-a,has-a,part-of

### is-a(汎化－特化)関係

「動物－犬」や「家電－テレビ」などのように「…は、○○である」で表される関係。

### part-of(集約－分解)関係

「コンピュータ－CPU」や「自転車－サドル」などのように「…は、○○の一部である」で表される関係。

### has-a(包含関係)

全体クラス has a 部分クラス。
