## HTML HEAD

ページが読み込まれてもウェブブラウザーには表示されない部分。  
構成情報などを記述する場所と覚えて差し支えないか。  

例:  
・`<title>` といった情報や CSS へのリンク (もし HTML を CSS で修飾したいならば)  
・独自のファビコンへのリンク  
・メタデータ (HTML を誰が書いたのかとかその HTML を表現する重要なキーワードなど)  

※メタデータ : データを説明するデータ  
`<meta charset="utf-8">`
